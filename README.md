# 🌐 Mini Browser

A simple web browser application built using **Node.js, Express.js, HTML, CSS, and JavaScript**.
This project allows users to enter a URL, fetch the webpage content from the server, and display it within the application.

---

## 🚀 Features

* 🔎 Enter any URL and fetch website content
* 🌐 Display webpage inside the app using iframe
* 📜 View browsing history
* ❌ Clear browsing history
* ⚠️ Handles invalid URLs with proper error messages
* ⏳ Loading spinner for better user experience

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **HTTP Client:** Axios

---


## ⚙️ How to Run Locally

### 1️⃣ Clone the repository

### 2️⃣ Navigate to backend

cd backend

### 3️⃣ Install dependencies

npm install

### 4️⃣ Start server

node server.js

### 5️⃣ Open frontend

Open `frontend/index.html` using Live Server
